# Python Magic Methods – Exam Project

Dette projekt er lavet som eksamensmateriale til Python valgfaget.

## Emne
Projektets emne er Python Magic Methods (Dunder Methods) og Python Data Model.

## Indhold
Projektet indeholder syv Python-eksempler, der tilsammen demonstrerer
hvordan objekter i Python kan styres og tilpasses ved hjælp af magic methods
og relaterede mekanismer.

Eksemplerne omfatter blandt andet:
1. Objekt-repræsentation og operator overloading (__repr__, __mul__)
2. Arv og operator overloading med Queue (__add__)
3. Python Data Model – sammenhængen mellem operatorer og metodekald (__add__ vs +)
4. Sammenligning af objekter (__eq__)
5. Brug af __len__ til brugerdefinerede objekter
6. Adgang til objekter via indeks (__getitem__)
7. Generatorer og programflow ved brug af yield

## Formål
Formålet med projektet er at demonstrere forståelse for objektorienteret
programmering i Python samt indsigt i, hvordan Python Data Model styrer
objekters opførsel gennem magic methods og beslægtede koncepter.
